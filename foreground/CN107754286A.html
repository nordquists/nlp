Abstract

The invention discloses a kind of table tennis system based on inertial navigation and sense of touch correction positioning.Its system by can recognize that table tennis impingement position table tennis table, be equipped with pressure sensor and photoelectric sensor table tennis net, be equipped with the table tennis bat of pressure sensor and gyroscope, have inertial navigation function table tennis and according to inertial navigation and sense of touch correct the server group of calculation position into.Each sensor disposed on table tennis table, net and racket obtains the shock data of table tennis, then data will be hit to send to the server for correcting calculation position according to inertial navigation and sense of touch, table tennis with inertial navigation function obtains table tennis data, then sends exercise data to the server that calculation position is corrected according to inertial navigation and sense of touch.The present invention solves the problems, such as that the existing table tennis system based on Internet of Things lacks and calculates table tennis position functions in real time.

Technical field
The invention belongs to smart table tennis Motion Technology field, is corrected more particularly to one kind based on inertial navigation and sense of touch The table tennis system of positioning.
Background technology
The technology being monitored at present to table tennis mainly has two classes, and one kind is computer vision analysis method, and one Kind is signal triggering determination methods.
Computer vision analysis method mainly gathers record during table tennis in real time by more high speed high-definition cameras Picture, each image in video recording is carried out to form data after target identification and space orientation, then by filtering and tracking calculates Table tennis track, it is computation complexity height the shortcomings that this method, it is high to the performance requirement of equipment.
Signal triggering determination methods are to be obtained by the piezoelectric transducer installed in table tennis table and net because being rattled The shock of ball and caused signal, the intensity and generation time of more multiple signals, calculate the impingement position and substantially of table tennis Movement locus, be that can not position the skyborne position of table tennis the shortcomings that the program, positional accuracy is low.
The content of the invention
The technical problems to be solved by the invention are that the existing table tennis system based on Internet of Things lacks calculating in real time The function of table tennis ball position, propose a kind of table tennis system based on inertial navigation and sense of touch correction positioning.
