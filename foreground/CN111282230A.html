CN111282230A - Table tennis bat with Bluetooth - Google Patents

Abstract

The invention provides a table tennis bat with Bluetooth, wherein the handle of the table tennis bat is internally provided with the Bluetooth, so that a control switch on the table tennis bat can be turned on when a user plays a table tennis, and the user can directly use the bat to talk after a phone call comes, thereby avoiding the influence on playing the table tennis due to frequent answering of the phone call.

Description

Table tennis bat with Bluetooth
Technical Field
The invention relates to a table tennis bat, in particular to a table tennis bat with Bluetooth.
Background
When people play table tennis, the mobile phone calls at the moment, and people can continue the game only after the game is paused and the mobile phone is put through. If the number of the incoming calls of the mobile phone is large, people cannot play table tennis normally, and people feel very worried.
Disclosure of Invention
The invention aims to provide a table tennis bat with Bluetooth, which is provided with the Bluetooth and can be connected with a mobile phone for conversation.
The invention comprises a table tennis bat, and adopts the technical scheme that: the handle of the table tennis bat is internally provided with Bluetooth, can be connected with a mobile phone and is provided with a control switch.
The invention has the beneficial effects that: because the Bluetooth is arranged in the handle of the table tennis bat, the control switch on the handle can be turned on when people play table tennis, and therefore people can directly use the bat to talk after mobile phone calls, and influence on playing the table tennis due to frequent answering of the phone is avoided.
