Abstract

PURPOSE: A rotary game board is provided to enable a user to enjoy a desired game by rotating a game board with different games on the front and back of the game board. CONSTITUTION: A rotary game board comprises two side boards(110) and a game board(120) which is installed between the side boards. The side board forms a protrusion unit so that the game board can be assembled and rotated. The game board is integrally formed and forms a groove unit in the central unit in the longitudinal direction. A penetration unit is able to be formed in the game board.

Rotating Games Table {Rotating Games Table}
The present invention relates to a rotatable game board, and more particularly, to a rotatable game board that is provided with different game boards on one side and the other side to rotate and enjoy a game desired by the user.
Recently, a table tennis game or a billiard game is a leisure sport, which has been enjoyed by people of various classes, not a specific class, and according to the popularization of table tennis and billiards, the demand for table tennis tables and billiard tables has increased greatly.
In general, a table tennis table and a billiard table are used to install different game tables to enjoy the game. However, the conventional game tables have an inconvenient problem because they have to assemble and separate the accessories used in the game complex.
Accordingly, there is an urgent need for the development of products that allow two games to be used in a table tennis table and a billiard table in a game table.
An object of the present invention has been proposed to solve the problems described above, it is an object of the present invention to provide a rotatable game board that allows users to easily assemble to enjoy a variety of games.
