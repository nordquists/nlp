Abstract

The present invention relates to a handle used in a tool having a body and a handle, and specifically, to a handle having a finger forward release prevention part (130) for preventing the hand of a user from sliding in the centrifugal direction when the handle (120) is gripped. According to the present invention, since the finger release prevention part (130) is provided on the handle, the hand of the user is prevented from slipping forward when the user grips the handle, and moreover, by providing a further tool forward release prevention part on the handle, the tool is prevented from slipping forward in reverse when the user grips the handle of the tool, and thus the handle of the tool is gripped without further shaking, thereby making use of the tool more comfortable and stable.
